# FinDashboard

Projeto desenvolvido para a disciplina de Front-End da FIAP.

## Descrição

O FinDashboard é uma interface de controle financeiro criada utilizando HTML, CSS e Tailwind CSS. A aplicação apresenta uma tela de dashboard com informações de saldo, entradas, saídas e transações recentes.

## Tecnologias Utilizadas

* HTML
* CSS
* Tailwind CSS

## Funcionalidades

* Dashboard financeiro
* Exibição de saldo atual
* Resumo de entradas e saídas
* Histórico de transações
* Layout responsivo
